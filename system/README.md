# System

This directory contains the reusable Obsidian learning-system structure:

- `templates/`: note and card templates
- `dashboards/`: Obsidian Bases used as review dashboards
- `pronunciation/`: entry points for accent and phoneme practice
- `vocabulary/`: vocabulary dashboard schema and notes
- `listening/`: listening-system placeholder and ListenKit boundary

The full private vault is not included. Use `examples/` for a small set of representative notes.
