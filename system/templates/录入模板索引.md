# 录入模板索引

复制下面的模板到对应文件夹后，再补完整字段。`review_stage` 统一使用 `day0 / day1 / day3 / day7 / day14 / day30 / day90 / day180 / mastered`。

## 每日学习清单模板

建议直接复制到当天笔记，不单独新建一套复习条目。

参考文件：[[学习系统/模板/每日学习清单模板]]

适用场景：

- 保存当天实际做了什么
- 记录卡点和断点
- 给明天留一个可直接开做的入口

## 课堂笔记模板

用于当天课堂原始笔记，和每日学习清单分开。

参考文件：[[学习系统/模板/课堂笔记模板]]

默认结构：

- `## 単語`
  - `### 通常語彙`
  - `### 漢字差分`
- `## 文法`
- `## 生活口語`
- `## 間違えた問題`

## アクセント對比卡规则

アクセント對比卡属于发音练习，不属于普通单词卡。用于同音或近音词需要通过アクセント区分意思、听感或朗读方式的情况。

默认位置：

- `学习系统/发音/アクセント/`

Frontmatter 关键字段：

```yaml
track: pronunciation
item_type: accent_contrast
issue_tags:
  - accent
  - minimal_pair
```

建卡规则：

- 一张卡收 8-12 组，标准按 10 组控制。
- 同一训练目标追加到同一张卡；超过 12 组时开下一张，例如 `アクセント対比_同音異義語_02.md`。
- 反复出错的单组可以拆成高优先级单独卡；原组卡保留简略索引即可。
- 保持在 `学习系统/发音/アクセント/` 下面，和 `学习系统/发音/练习/` 并列；`练习/` 只放句子发音、录音和跟读类练习。
- 如果问题是词义不清，才进入 `学习系统/课堂复习/词汇`；如果问题是アクセント听辨或朗读，放在发音练习。

## 音素對比卡规则

音素對比卡用于清音/浊音、送气、声带振动、口腔动作等最小发音差异。它不放在句子练习目录。

默认位置：

- `学习系统/发音/音素/`

Frontmatter 关键字段：

```yaml
track: pronunciation
item_type: phoneme_contrast
issue_tags:
  - 清浊音混淆
```

建卡规则：

- 规则型、辨别型发音卡放在 `学习系统/发音/音素/`。
- 句子整体语调、停顿、录音复盘放在 `学习系统/发音/练习/`。
- アクセント听辨或朗读问题放在 `学习系统/发音/アクセント/`。
