# アクセント対比：同音異義語 01

## 目标

用アクセント区分同音或近音词，不只记意思。练习时先读平假名和アクセント，再说出对应意思。

## 对比表

あめ、はし、いつか、むし、さとう、あつい、かみ、かむ、いし、かえる

| 读音 | アクセント | 意思 | 练习提示 |
| :--- | :--- | :--- | :--- |
| あめ | ⓪ | 飴 | 平板型，后续助词保持高。 |
| あめ | ① | 雨 | 头高型，第一拍后下降。 |
| はし | ⓪ | 橋 | 平板型。 |
| はし | ① | 箸 | 头高型。 |
| いつか | ⓪ | 5日 | 平板型，作为日期读。 |
| いつか | ① | 何時か | 头高型，表示“改天 / 什么时候”。 |
| むし | ⓪ | 虫 | 平板型。 |
| むし | ① | 無視 | 头高型。 |
| さとう | ② | 砂糖 | 第二拍后下降。 |
| さとう | ① | 佐藤 | 第一拍后下降。 |
| あつい | ⓪ | 厚い | 平板型。 |
| あつい | ② | 暑い / 熱い | 第二拍后下降。 |
| かみ | ⓪ | 髪 / 紙 | 平板型。 |
| かみ | ① | 神 | 头高型。 |
| かむ | ⓪ | 鼻をかむ | 平板型，擤鼻涕。 |
| かむ | ① | 噛む | 头高型，咬。 |
| いし | ⓪ | 石 | 平板型。 |
| いし | ① | 医師 | 头高型。 |
| かえる | ⓪ | 蛙 | 平板型。 |
| かえる | ① | 帰る | 头高型。 |

## 练习步骤

1. 先遮住意思，只看读音和アクセント读一遍。
2. 再遮住アクセント，只看读音判断意思。
3. 最后任选 5 组录音，回听下降位置是否稳定。
